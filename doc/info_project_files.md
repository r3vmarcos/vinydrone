# Arquivos do Projeto

## Raiz

- `package.json`: scripts e dependências do projeto React, TypeScript, Vite e Tailwind CSS.
- `package-lock.json`: trava de versões instalada pelo npm.
- `index.html`: entrada HTML do Vite.
- `vite.config.ts`: configuração principal do Vite.
- `tailwind.config.ts`: configuração principal do Tailwind CSS.
- `postcss.config.cjs`: configuração do PostCSS.
- `tsconfig*.json`: configurações TypeScript.
- `.gitignore`: arquivos e pastas ignorados pelo Git.

## `src`

- `src/components/sections`: seções principais da landing page.
- `src/components/ui`: componentes reutilizáveis de interface.
- `src/data/siteData.ts`: dados editáveis de navegação, serviços, diferenciais e equipamentos.
- `src/utils`: utilitários do projeto.
- `src/assets/photos`: imagens utilizadas na galeria e equipamentos.
- `src/assets/ui`: logos e ícones.
- `src/assets/videos`: vídeo local usado no hero.

## Documentação

- `prompts.md`: histórico cumulativo resumido de prompts relevantes.
- `doc/historico_chat.md`: registro cumulativo de pedidos, ações e validações.
- `doc/changelog.md`: histórico objetivo de alterações.
- `doc/info_project_files.md`: mapa dos arquivos relevantes do projeto.
